ZABBIX is an all-in-one 24x7 monitoring solution without high cost. ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. The full licence is available at http://www.gnu.org/copyleft/gpl.txt.

ZABBIX can be installed on any UNIX platform. It acts as a server that monitors client systems running on different platforms as well as other network devices. ZABBIX has a quick install and can be up and running in less than 15 minutes.

The 1.1beta1 release introduces the following functionality:

  • basic configuration management
  • new function fuzzytime()
  • support of macros $1,..,$9 in item descriptions
  • new service check dns[ip, domain]
  • new configuration parameter DBPort
  • if trigger cannot be evaluated, an error message is shown in GUI
  • time format for log files
  • new naming convention for most of monitored parameters
  • monitoring of file modification/access/creation times
  • new parameter system.localtime
  • parameter Hostname is optional. Real hostname is used, if not specified
  • German and French translations of WEB GUI

The release also fixes the following issues:

  • never logout option in user profile
  • hosts-templates linkage (manipulation of actions, etc)
  • world access to ZABBIX log files
  • misc usability fixes for PHP interface
  • fixed processing of W32 performance counters. Problem with long computer names
  • Win32 agent, processing of long messages of event logs
  • update of trigger valus in certain situations

The release can be used in production environment.
