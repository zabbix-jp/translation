ZABBIX SIA is proud to announce the availability of ZABBIX 1.3.2.

ZABBIX is an enterprise-class open source distributed monitoring solution.

ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. The full license is available at http://www.gnu.org/copyleft/gpl.txt.

This document contains the release notes for ZABBIX 1.3.2. The release supports MySQL, PostgreSQL and SQLite only. Support of Oracle will be restored in later releases.

Do not use this release in production! It is for beta testing only.

The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

What's New in 1.3.2

Restored support of PostgreSQL

Support of PostgreSQL has been restored. Use PostgreSQL 8.x for better performance and more flexible autovacuum.

Added editor of simple trigger expressions

The editor allows creation of simple trigger expressions without thinking of syntax. Try it and please let us know what you think.

Misc functionality

  • ZABBIX Manual 1.4 release 2
  • removed support of server parameter DisablePinger, use StartPingers
  • fixed PostgreSQL issues
  • fixed disconnect from terminal (redirection of stderr/stdout)
  • improved support of SQLite
  • other improvements and bug fixes

Installation and Upgrade Notes

Installation

See the Installation Guide http://www.zabbix.com/manual/v1.4/install.php for full details. See http://www.zabbix.com/manual/v1.4/install_source_web.php for information about Installation Wizard.

Upgrade

No upgrade script are provided for migration from 1.1.x. Please do not upgrade! The release may or may not work on your system. It has been tested on a very limited number of systems.

Pre-release testing

ZABBIX Server

ZABBIX Server has been compiled and tested on the following platforms:

  • Ubuntu Linux, AMD64, kernel 2.6.11, MySQL 4.x
  • Ubuntu Linux, Intel, kernel 2.6.15, MySQL 5.0.22

ZABBIX Agents

ZABBIX Agents have been compiled and tested on the following platforms:

  • Linux 2.6.11 (Ubuntu)
  • Windows XP
  • Windows XP 64bit

Note that agents are available pre-compiled from http://www.zabbix.com/download.php for a limited number of versions and platforms.
