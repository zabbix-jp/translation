ZABBIX SIA is proud to announce the availability of ZABBIX 1.1.

ZABBIX is an all-in-one 24x7 monitoring solution without high cost. ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. The full license is available at http://www.gnu.org/copyleft/gpl.txt.

This document contains the release notes for ZABBIX 1.1. The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

What's New in 1.1

ZABBIX Server

  • One server process
  • Support of SNMPv3
  • Monitoring of log files
  • Monitoring of Windows events
  • Auto-registration
  • Active agent
  • Trigger acknowledgements
  • Support of Oracle

Visualisation

  • More flexible graphs
  • Multi level maps
  • Maps with backgrounds
  • Overview screens

High-performance agents

  • AIX, HP-UX, Linux, Solaris, Tru64
  • *BSD
  • Windows XP, 2000, 2003
  • Novell
  • Max OS/X

GUI

  • Fully rewritten
  • Audit log
  • Localisation (English, French, German, etc)
  • Value mapping
  • User-defined tactical overview screens
  • More flexible user screens

Templates

  • One template per thousand of hosts

Pro-active monitoring

  • Automatic execution of remote commands
  • Flexible definition of actions

Aggregate monitoring

  • Monitoring of a group of hosts as s signle host

Configuration management

  • Hardware details about monitored devices

...and more!

Installation

Pre-release testing

ZABBIX Server

ZABBIX Server has been compiled and tested on the following platforms:

  • Ubuntu Linux, AMD64, kernel 2.6.11, MySQL 4.x
  • RedHat AS2 Linux, Intel, kernel 2.4.x, Oracle 9.2.0.4

ZABBIX Agents

ZABBIX Agents have been compiled and tested on the following platforms:

  • AIX 5.2
  • FreeBSD 4.11, 5.4
  • HP-UX 11.00
  • Linux 2.6.11 (Ubuntu)
  • NetBSD 2.0
  • OS/X 10.2
  • Solaris 5.9
  • Tru64 5.1B

Note that agents are available pre-compiled from http://www.zabbix.com/download.php.
