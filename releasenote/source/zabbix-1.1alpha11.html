ZABBIX is an all-in-one 24x7 monitoring solution without high cost. ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. The full licence is available at http://www.gnu.org/copyleft/gpl.txt.

ZABBIX can be installed on any UNIX platform. It acts as a server that monitors client systems running on different platforms as well as other network devices. ZABBIX has a quick install and can be up and running in less than 15 minutes.

The 1.1alpha11 release introduces modularization of source code along with full support of autoconf framework. The release also includes:

  • support of active checks by Win32 agent
  • new XML based communication protocol (old protocol is still supported)
  • support of localisation
  • support of user profiles
  • fixed operations with complex triggers for templates
  • fixed selection of IT Services calculation algorithm
  • deleted hosts will not be visible
  • support of row span (screens)
  • many other changes

Please note this release does not support escalation of notifications.

It's not recommended to use this release in production environment.

Please, see ChangeLog for complete list of changes.
