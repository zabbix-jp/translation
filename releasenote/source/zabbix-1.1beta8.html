This document contains the release notes for ZABBIX version 1.1beta8. The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

The release introduces feature freeze for 1.1 series of releases. No more new functionality will be implemented.

What's New

Hierarchical system maps

A multi-level system maps can be created. An icon may show state of all elements of a lower level map. An icon not linked ot any host can be displayed on a map.

Acknowledgements for triggers

A trigger can be acknowledged and history of the acknowledgements and administrators' comments is saved.

Many improvements for screens

A screen may contain the following new elements: Clock, Server Info, Data Overview, Triggers Overview, Host Info, external URL and Triggers Info. ZABBIX administrators are free to build any tactical overview screens!

Added horizontal and vertical alignment for screen elements. Additionaly a screen may consist of other screens.

Remote command execution.

In addition to notifications, ZABBIX can be configured to execute remote commands on a remote host or a group of hosts in case of pre-defined events.

Improved Trigger Overview screen.

Elemnts of the overview screen will blink in case of recent status change.

Fixes

  • Fixed inefficient check of user permissions.
  • Other fixes.

What's next

Release of ZABBIX 1.1 is really close.

Please use ZABBIX Forums to inform ZABBIX Development Team about any issues found in the beta.

Installation and Upgrade Notes

Installation

See the Installation Guide http://www.zabbix.com/manual/v1.1/install.php for full details.

Upgrade

In order to upgrade from previous release, execution of database patches required. See Upgrade Guide http://www.zabbix.com/manual/v1.1/config_upgrading.php for full details on upgrade procedure.

Pre-release testing

ZABBIX Server

ZABBIX Server has been compiled and tested on the following platforms: Ubuntu Linux, AMD64, kernel 2.6.11

ZABBIX Agents

ZABBIX Agents have been compiled and tested on the following platforms:

  • AIX 5.2
  • FreeBSD 4.11, 5.4
  • HP-UX 11.00
  • Linux 2.6.11 (Ubuntu)
  • NetBSD 2.0
  • OS/X 10.2
  • Solaris 5.9
  • Tru64 5.1B

Note that agents are available pre-compiled from www.zabbix.com/download.php. The agents are compatible with ZABBIX v1.0 and all releases in 1.1 series.

Use in production environments

ZABBIX 1.1beta8 is BETA software, therefore we cannot recommend it for immediate use in production. Although tested throughout our release cycle, this release introduces many changes and does not have the benefit of use in production environments that ZABBIX v1.0 has and therefore should be treated with the same level of caution as a new product.
