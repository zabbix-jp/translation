ZABBIX SIA is proud to announce the availability of ZABBIX 1.6.2.

ZABBIX is an enterprise-class open source distributed monitoring solution. ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. A complete text of the license is available at http://www.gnu.org/copyleft/gpl.txt.

It is highly recommended to upgrade from 1.6 and 1.6.1.

This document contains the release notes for ZABBIX 1.6.2.

The following sections describe the release in details and provide late-breaking or other information that supplements its main documentation.

Important

Starting from this release ZABBIX sends notification messages to the users having read access to a host, which generated the event only.

What's New in 1.6.2

Better support of SNMP data

Added support of quoted numbers.

New supported checks

Added support of system.swap.size and net.tcp.listen under FreeBSD, system.boottime under Solaris, system.cpu.load under Solaris 2.6, vm.memory.size under AIX 6.x. proc.num and proc.mem are fully supported under Linux, Solaris and AIX.

Installation and Upgrade Notes

Installation

See ZABBIX Manual for full details.

Upgrade

Recompile ZABBIX binaries and update front-end PHP files, no database patching needed. See ZABBIX Manual for a detailed upgrade procedure.

Pre-release testing

ZABBIX Server

ZABBIX Server has been tested on the following platforms:

  • Ubuntu Linux, AMD64, kernel 2.6.11, MySQL 4.x
  • Ubuntu Linux, Intel, kernel 2.6.15, MySQL 5.0.22

ZABBIX Agents

ZABBIX Agents have been compiled and tested on the following platforms:

  • AIX 5.2
  • FreeBSD 4.x, 5.x, 6.x
  • HP-UX 10.x, 11.x
  • Linux 2.4.x, 2.6.x
  • Linux CentOS
  • NetBSD 2.0
  • OS/X 10.2
  • Solaris 8, 9, 10
  • Tru64 5.1B
  • Windows XP, 2000, 2003, Vista

Note that agents are available pre-compiled from http://www.zabbix.com/download.php for a limited number of versions and platforms.
