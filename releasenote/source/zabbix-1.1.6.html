ZABBIX SIA is proud to announce the availability of ZABBIX 1.1.6

ZABBIX is an all-in-one 24x7 monitoring solution without high cost. ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. The full license is available at http://www.gnu.org/copyleft/gpl.txt.

This document contains the release notes for ZABBIX 1.1.6. The release does not introduce new functionality, it fixes all known issues of ZABBIX 1.1.5.

The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

What's New in 1.1.6

Fixed

  • fixed displaying of graphs under Oracle
  • improved 'perf_counter[*]' for WinXX
  • fixed 'service_state[*]' for WinXX
  • fixed 'History of events' and 'Server info' screen components
  • fixed stderr/stdout redirection during logrotation
  • fixed macros substitution for messages
  • fixed truncated alert messages
  • minor fixes of permission system
  • minor SQL related fixes for PostgreSQL
  • fixed trigger expressions having characters ':' or '-'
  • fixed support of character '-' in host names

Installation and Upgrade Notes

Installation

See the Installation Guide http://www.zabbix.com/manual/v1.1/install.php for full details.

Upgrade

You may upgrade from ZABBIX 1.1, 1.1.1, 1.1.2, 1.1.3, 1.1.4, 1.1.5 directly to 1.1.6. It requires recompilation of ZABBIX Server and installation of new front-end only. Optionally you may upgrade your agents. Upgrade if Windows agents is recommended.

In order to upgrade from previous releases, no execution of database patches required. See Upgrade Guide http://www.zabbix.com/manual/v1.1/install_upgrading.php for full details on upgrade procedure.
