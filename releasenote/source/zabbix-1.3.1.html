ZABBIX SIA is proud to announce the availability of ZABBIX 1.3.1.

ZABBIX is an enterprise-class open source distributed monitoring solution. ZABBIX is released under the GPL, thus it is free of charge for both commercial and non-commercial use. The full license is available at http://www.gnu.org/copyleft/gpl.txt.

This document contains the release notes for ZABBIX 1.3.1. The release supports MySQL and SQLite only. Support of Oracle and PostgreSQL will be restored in next beta.

Do not use this release in production! It is for early beta testing only.

The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

What's New in 1.3.1

XML Import/Export

New XML based import/export allows manipulation with hosts, items and triggers making it an excellent choice for sharing common templates. Old bulkloader has been removed.

Support of SQLite3

SQLite is an excellent choice for running zero-administration ZABBIX nodes on an embedded hardware. Not recommended for large setups due to inefficient locking mehanism.

ZABBIX Manual in PDF format

Download, read and print ZABBIX manual! Find it in directory doc/ of ZABBIX source distribution.

Transactional approach for ZABBIX Server

Use of transactions improves performance of ZABBIX, it also maintains better integrity and security of ZABBIX data.

New configuration parameters for ZABBIX server

  • StartPingers
  • StartPollersUnreachable

New definition of some parameters of ZABBIX server

  • StartPollers
  • StartTrappers

Misc functionality

  • better editing of custom graphs
  • support of '-n' flag for ZABBIX Server: change Node ID
  • improved error displaying
  • added 'Host Group' element to graphs
  • other improvements and bug fixes

Installation and Upgrade Notes

Installation

See the Installation Guide http://www.zabbix.com/manual/v1.4/install.php for full details. See http://www.zabbix.com/manual/v1.4/install_source_web.php for information about Installation Wizard.

Upgrade

No upgrade script are provided for migration from 1.1.x. Please do not upgrade! The release may or may not work on your system. It has been tested on a very limited number of systems.

Pre-release testing

ZABBIX Server

ZABBIX Server has been compiled and tested on the following platforms:

  • Ubuntu Linux, AMD64, kernel 2.6.11, MySQL 4.x
  • Ubuntu Linux, Intel, kernel 2.6.15, MySQL 5.0.22

ZABBIX Agents

ZABBIX Agents have been compiled and tested on the following platforms:

  • Linux 2.6.11 (Ubuntu)
  • Windows XP
  • Windows XP 64bit

Note that agents are available pre-compiled from http://www.zabbix.com/download.php for a limited number of versions and platforms.
